# Craftom School

MVP למערכת לומדה לקורס Craftom לכיתה ד׳.

## מה קיים כרגע

- התחברות והרשמה
- תפקידים בסיסיים: admin, teacher, student
- דף בית תלמיד
- דף בית מורה
- עורך שיעורים למורה/אדמין
- תבנית שיעור ריקה למורה
- דוח התקדמות לתלמיד ולמורה
- תצפיות מורה למדדי שיתוף פעולה
- מודל קורס של 15 מפגשים
- כיתות וקודי כיתה
- צוותים וקודי הצטרפות לצוות
- Preview סטטי חיצוני

## הרצה מקומית

```bash
npm start
```

ברירת מחדל:

```text
http://127.0.0.1:3081
```

אפשר לבחור פורט:

```bash
PORT=3082 npm start
```

## הערות MVP

- הנתונים נשמרים בקובץ `data/state.json`, והוא לא נכנס לגיט.
- מורים חדשים מוגדרים כ־pending.
- מורה יכול ליצור כמה כיתות ולקבל קוד הרשמה לכל כיתה.
- תלמיד יכול להירשם רק עם קוד כיתה תקין, ומשויך לכיתה אחת בלבד.
- עורך השיעורים שומר גרסאות ערוכות ב־`data/state.json` מעל תבנית ברירת המחדל של 15 המפגשים.
- זה עדיין לא מחובר ל־Minecraft/Craftom בפועל.
