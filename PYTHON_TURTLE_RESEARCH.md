# מחקר קצר: Blockly + Python Turtle לילדי כיתה ה׳ מתחילים

## מסקנה
לילדים שמגיעים בלי שנת הבסיס של כיתה ד׳, כדאי להתחיל מ־Blockly שמייצר Python גלוי, עם רובוט מצייר בסגנון Turtle. זה מוריד עומס תחבירי אבל עדיין מחבר אותם לשפה אמיתית.

## מקורות ורציונל
- Python docs מתארים את turtle כ״רובוט קטן עם עט״ שמספק פידבק חזותי מיידי, ולכן מתאים במיוחד להיכרות ראשונה עם תכנות ואינטראקציה עם תוכנה.
- Grok Academy מפעילים מסלול Blockly Turtle לגילאי Years 5–6, סביב 8–15 שעות: קווים, פניות, צבעים, לולאות, תנאים ופתרון בעיות פשוטות.
- קורסי Python Turtle לילדים מדגישים יצירה וציור: forward, turn, color, patterns — במקום להתחיל מהדפסות טקסט מופשטות.

## עקרונות לפיתוח הלומדה שלנו
1. **קודם בלוקים, אחר כך טקסט** — התלמיד גורר פקודות ומיד רואה Python בתחתית המסך.
2. **רובוט מצייר = פידבק מיידי** — כל פעולה מייצרת תנועה או ציור ברור.
3. **מסלול יסודות הדרגתי**:
   - רצף פקודות
   - מספרים ופרמטרים
   - זוויות וצורות
   - לולאות `for`
   - צבע/עט/מצב
   - משתנים
   - תנאים
   - קלט
   - פונקציות
   - פרויקט ציור אישי
4. **לא להעמיס HTML/CSS/JS בשלב ראשון** — זה מסלול המשך, לא נקודת פתיחה לילדים חדשים.

## MVP שנבנה עכשיו
`python-turtle.html`:
- Blockly בעברית
- בלוקים: קדימה, ימינה, שמאלה, צבע, עובי עט, עט למעלה/למטה, חזור N פעמים, הערה
- קוד Python חי בתחתית
- Canvas עם רובוט 🐢 שמריץ את הציור
- 5 משימות פתיחה: פקודות, ריבוע, צבע/עובי, עט למעלה/למטה, כוכב
