# WebCode Lab — תוכנית 30 שיעורים

## רעיון פדגוגי

המטרה היא לא להשאיר את הילדים ב־Blockly לנצח, אלא להשתמש בו כגשר בטוח:

1. **שיעורים 1–4 — חוויה לפני תחביר**  
   גוררים בלוקים, רואים עמוד חי, מבינים HTML/CSS/JS דרך תוצאה.

2. **שיעורים 5–12 — משחקים עם לוגיקה דרך בלוקים**  
   תנאים, משתנים, ניקוד, טיימר, חיים, מסכים ופרויקט משחק ראשון.

3. **שיעורים 13–18 — בלוק שמייצר קוד אמיתי**  
   עדיין גוררים, אבל כל בלוק מדגיש שורת קוד. הילדים מתחילים לערוך ערכים קטנים בקוד שנוצר.

4. **שיעורים 19–24 — גרירת קוד אמיתי**  
   עוברים מבלוקים צבעוניים ל“כרטיסי קוד”: שורות JavaScript/HTML/CSS אמיתיות שגוררים, מסדרים ומשלימים.

5. **שיעורים 25–30 — כתיבה אמיתית מודרכת**  
   פחות גרירה, יותר השלמה, תיקון וכתיבה עצמאית. בסוף: פרויקט Web אישי אמיתי.

הכלל: **קודם תוצאה שילד רוצה להראות, אחר כך מושג, אחר כך קוד.**

## שלב א׳ — Blockly חי: בונים, מעצבים, מפעילים, מקשיבים

### שיעור 1 — בונים עמוד עם Blockly אמיתי
- תוצר: עמוד אישי עם כותרת, פסקה, אימוג׳י וכפתור.
- מושגים: HTML, אלמנטים, שרשרת בלוקים.
- מצב קוד: הצצה בלבד.

### שיעור 2 — סטודיו עיצוב
- תוצר: אותו עמוד מקבל צבעים, צורה, צל ו־hover.
- מושגים: CSS, background, border-radius, box-shadow, hover.
- מצב קוד: בחירת בלוק מדגישה CSS שנוצר.

### שיעור 3 — כפתורים שמפעילים קסמים
- תוצר: כפתור משנה הודעה, אימוג׳י, רקע ומצב קסם.
- מושגים: onclick, function, textContent, style, classList.toggle.
- מצב קוד: בלוק פעולה מדגיש JavaScript.

### שיעור 4 — העמוד שמקשיב לי
- תוצר: מחולל ברכות מצחיקות לפי שם ותחביב.
- מושגים: input, id, value, שרשור טקסט.
- מצב קוד: הילדים רואים איך הקוד קורא את מה שהם כתבו.

## שלב ב׳ — לוגיקה ומשחקים דרך בלוקים

### שיעור 5 — חידון כן/לא
- תוצר: חידון עם תשובה נכונה ושגויה.
- מושגים: if / else, תנאי, משוב.
- מצב קוד: מזהים את מבנה התנאי.

### שיעור 6 — ניקוד ומשתנים
- תוצר: חידון עם ניקוד עולה.
- מושגים: variable, score, עדכון מסך.
- מצב קוד: מזהים `let score = 0` ו־`score = score + 1`.

### שיעור 7 — משחק קליקים ראשון
- תוצר: משחק שבו לוחצים כדי להגיע ליעד.
- מושגים: target, click event, win condition.
- מצב קוד: משנים מספר יעד דרך בלוק.

### שיעור 8 — משחק עם טיימר
- תוצר: אתגר קליקים בזמן מוגבל.
- מושגים: timeLeft, setInterval, סיום משחק.
- מצב קוד: מזהים זמן שחוזר כל שנייה.

### שיעור 9 — מכשולים ופסילה
- תוצר: משחק עם חיים, כוכבים ומכשולים.

### שיעור 10 — כוח מיוחד לדמות
- תוצר: כוח חד־פעמי / בוסט / מגן.
- מושגים: state, boolean, cooldown בסיסי.

### שיעור 11 — מסכי משחק
- תוצר: מסך פתיחה, משחק, ניצחון והפסד.
- מושגים: screen, class active, show/hide.
- מצב קוד: מזהים פונקציית מעבר מסכים.

### שיעור 12 — מיני־פרויקט ראשון
- תוצר: משחק Web קטן מחובר: ניקוד + זמן + חיים + מסכים.
- מושגים: חיבור מערכות.
- מצב קוד: בעיקר בלוקים, הצגת קוד בסוף.

## שלב ג׳ — בלוקים שמובילים לעריכת קוד

### שיעור 13 — המעבדה של הקוד שנוצר
- תוצר: ילדים בוחרים בלוק ורואים בדיוק את שורת הקוד שלו.
- מושגים: mapping בין בלוק לקוד.
- מעבר: מתחילים לערוך ערכים קטנים בקוד: טקסט, צבע, מספר.

### שיעור 14 — משנים HTML אמיתי בזהירות

### שיעור 15 — משנים CSS אמיתי בזהירות
- תוצר: עיצוב אישי דרך עריכת CSS ישיר.
- מושגים: property/value, colors, px, selector.
- מעבר: “גרירת בלוק” לצד “שינוי ערך בקוד”.

### שיעור 16 — משנים JavaScript אמיתי בזהירות
- תוצר: כפתור עם הודעות/תגובות שונות.

### שיעור 17 — מספרים שמשנים משחק
- תוצר: משחק שבו שינוי מספרים משנה קושי.
- מושגים: constants, target, speed, lives.
- מעבר: עריכת מספרים בקוד במקום בלוק.

### שיעור 18 — תיקון באגים ראשונים
- תוצר: “חדר בריחה של באגים” — מתקנים קוד שבור.
- מושגים: שגיאות נפוצות: מרכאות, סוגריים, id לא תואם.
- מעבר: הילדים לומדים לקרוא הודעת שגיאה פשוטה.

## שלב ד׳ — גרירת קוד אמיתי במקום בלוקים

בשלב הזה ה־toolbox כבר לא מכיל רק בלוקים כמו “כפתור”, אלא **כרטיסי קוד אמיתי**.  
למשל כרטיס:

```js
const name = document.getElementById("nameInput").value;
```

הילד גורר שורת קוד אמיתית למקום הנכון, ואז משנה חלק קטן ממנה.

### שיעור 19 — כרטיסי HTML אמיתי
- תוצר: בניית עמוד מכרטיסי HTML אמיתיים.
- מושגים: תג פתיחה/סגירה, nesting פשוט.
- מעבר: גוררים `<section>`, `<h1>`, `<p>`, `<button>`.

### שיעור 20 — כרטיסי CSS אמיתי
- תוצר: עיצוב עמוד מכרטיסי CSS אמיתיים.
- מושגים: selector, declaration block, property/value.
- מעבר: גוררים `body { ... }`, `.card { ... }`, `button:hover`.

### שיעור 21 — כרטיסי JavaScript אמיתי
- תוצר: כפתור שמשנה הודעה מכרטיסי JS.
- מושגים: function, variable, assignment.
- מעבר: גוררים שורות לתוך function בסדר נכון.

### שיעור 22 — מסדרים קוד בסדר נכון
- תוצר: משחק קטן עובד רק אם השורות בסדר הנכון.
- מושגים: flow, sequence, initialization before use.
- מעבר: פאזל קוד אמיתי.

### שיעור 23 — משלימים חורים בקוד
- תוצר: מחולל או חידון עם “חורים” להשלמה.
- מושגים: fill-in-the-blank: id, text, number, operator.
- מעבר: לא גוררים הכל — משלימים חלקים.

### שיעור 24 — משנים חוקי משחק בקוד אמיתי
- תוצר: משחק קליקים/ניקוד שבו הילדים משנים חוקים בעצמם.
- מושגים: if, score, target, lives.
- מעבר: עריכת תנאי ומספרים בקוד.

## שלב ה׳ — כתיבת קוד מודרכת

### שיעור 25 — כותבים HTML ראשון לבד
- תוצר: עמוד אישי בלי Blockly.
- מושגים: מבנה HTML בסיסי בתוך אזור מוגן.
- תמיכה: snippets בצד, בדיקה אוטומטית.

### שיעור 26 — כותבים CSS ראשון לבד
- תוצר: עיצוב אישי מלא לעמוד קיים.
- מושגים: selector, color, spacing, border, hover.
- תמיכה: autocomplete/רמזים, לא בלוקים.

### שיעור 27 — כותבים function ראשון לבד
- תוצר: כפתור שמפעיל פונקציה שכתבו בעצמם.
- מושגים: function name, body, textContent.
- תמיכה: שלד מוכן, משלימים בפנים.

### שיעור 28 — בונים חידון בקוד
- תוצר: חידון עם input, if/else ומשוב.
- מושגים: value, comparison, if/else.
- תמיכה: חלק מהקוד מוכן, הילדים כותבים את התנאי והמשוב.

### שיעור 29 — פרויקט אישי בתכנון
- תוצר: תכנון פרויקט: אתר/משחק/מחולל.
- מושגים: אפיון קטן, מסכים, פעולות, נתונים.
- תמיכה: תבנית תכנון + בחירת starter.

### שיעור 30 — פרויקט סיום: Web App קטן
- תוצר: פרויקט אישי עובד: משחק / מחולל / חידון / דף אינטראקטיבי.
- מושגים: שילוב HTML + CSS + JS.
- מצב קוד: כתיבה אמיתית, עם snippets ורמזים בלבד.
- סיום: תערוכה, שיתוף קישור, רפלקציה: “מה כתבתי לבד?”

## המלצה שלי

כן — להגיע לכתיבת קוד אמיתי בסוף זה הכיוון הנכון.  
אבל לא לקפוץ לשם מוקדם מדי.

המעבר הכי בריא הוא:

**בלוק צבעוני → בלוק שמדגיש קוד → כרטיס קוד אמיתי → חור בקוד → כתיבה חופשית.**

ככה הילדים לא מרגישים ש“לקחו להם את Blockly”, אלא שהם גילו שהוא היה גלגלי עזר.
